Insync-Coding

    "Partner up and sync your code at 'Insync Code'! This event is all about the seamless collaboration of two minds in the world of coding. Get ready to tackle challenges as a dynamic duo!"

    Time : 1:30 PM ~ 2:30 PM

    Time Management

  • Participants must adhere to the time limits set for each round.

  • Failure to do so may result in penalties
